# Multi-Agent-Research-Assistant
Using LLaMA 2 / Mistral (via Ollama) &amp; Streamlit

An AI-powered research system that uses multiple specialized agents to:
- Search information
- Summarize findings
- Check factual accuracy
- Generate executive-ready reports

## Agents
- Search Agent
- Summarizer Agent
- Fact Checker Agent
- Report Generator Agent

## How to Run
1. Pull model: ollama pull mistral
2. Start model: ollama run mistral
3. Launch app: streamlit run frontend.py
